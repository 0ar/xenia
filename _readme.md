* [Install Ruby 2.2+ and DevKit](http://rubyinstaller.org/downloads).
* `gem update --system`
* Extract DevKit
* `ruby dk.rb init`
* `ruby dk.rb install`
* `gem install bundler`

From the repository root:

```
bundle install
bundle exec jekyll serve
```
